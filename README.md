# Cineflex

<!-- <img src="/assets/my-wallet-usage.gif" /> -->

Try it out now at https://cineflex-alpha.vercel.app/

## About

With a focus on presenting a simulation environment for choosing films and seats in a completely streamlined way, this application has interfaces for selecting films, sections, seats and the ending screem.
